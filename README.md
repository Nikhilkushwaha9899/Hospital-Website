🏥 About the Project

MediClub is a static hospital website created to showcase essential hospital-related information such as services, departments, doctors, and contact details.
This project does not include backend functionality or databases.

🛠️ Technologies Used

HTML5 – Structure of the website

CSS3 – Styling and layout

✨ Features

-> Responsive and clean UI design

-> Home page with hospital overview

-> Sections for services and medical facilities

-> Doctor or staff information section

-> Contact and appointment-related layout (UI only)

-> Easy-to-read and well-structured code

Suitable for beginners learning HTML & CSS
